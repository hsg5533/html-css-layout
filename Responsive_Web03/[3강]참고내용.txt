● 가변그리드(Fluid Grid) ●
- px : 고정값 / 화면사이즈에 변화와 상관없이 고정되는 값  
- % :  비율값 /  화면사이즈에 따라 차지하는 비율로 유동적인 값  

※ 가로 폭에 해당하는 부분을 계산하여 표현 

 { 공식 } 
  (요소의 너비 / 요소를 감싸는 부모박스의 너비) x 100 = 가변 너비 %
  margin+padding 동일하게 표현가능


 위의 공식에 해당하지 않는 부분 
 100%을 꽉채우면서 분배를 해야하는 경우 공식에 대한 계산이 
 아닌 나눌 분배값에 해당하여 계산합니다.  

   ex) 3개의 박스를 표현하고 싶다.  
   100% / 3 = 33.3%
